# 🦇 Valencia CF DBN — Repositorio Kodi

Repositorio oficial del skin **VCF DBN** para Kodi 21 (Omega).

## Instalación

### 1. Añadir fuente en Kodi
`Ajustes → Sistema → Gestor de archivos → Añadir fuente`

```
https://danielet21.github.io/repository.vcf.dbn/
```
Nombre: `VCF DBN`

### 2. Instalar repositorio
`Complementos → Instalar desde zip → VCF DBN → repository.vcf.dbn-1.0.0.zip`

### 3. Instalar skin desde repositorio
`Complementos → Instalar desde repositorio → Valencia CF DBN Repository → Skins → VCF DBN Skin`

## Contenido
